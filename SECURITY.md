# Security Policy

This repository documents static evidence of backdoor-style capabilities in DogAntiCheat.

Please do not open issues or pull requests containing bypass methods, spoofing methods, or instructions for evading anti-cheat checks.

Acceptable contributions:

- additional static evidence
- corrected decompilation
- clearer variable naming
- native import/call-graph analysis
- privacy-impact analysis
- safe removal and containment advice

Not acceptable:

- bypass tutorials
- server verification forgery
- disabling detection logic
- cheat enablement
- weaponized payloads
